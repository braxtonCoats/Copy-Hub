# Figma Copy Repository Plugin

This is a code bundle for Figma Copy Repository Plugin. The original project is available at https://www.figma.com/design/IBjZFZrPPMVMSjtdF0TmWe/Figma-Copy-Repository-Plugin.

## Development

### Prerequisites

- Node.js (v20 or higher recommended)
- npm

### Local Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

The production build will be output to the `dist/` directory.
